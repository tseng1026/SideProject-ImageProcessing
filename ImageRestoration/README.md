# SideProject - ImageRestoration

## Basic Execution
- **Platform:** Unix (Macbook)
- **Language:** Python3
- **Environment:** CPU
- **Usage:**
	- ``python ImageDenoising.py -i <input dir> -o <output dir>``
- **Requirements:**
	- python3.6
	- glob
	- attr
	- opencv ``pip install opencv-python``
	- numpy ``pip install numpy``

## Algorithm
- low pass filtering
- median filtering
- pseudomedian filtering
